🅿️ Hiace 停車筆記

優先選擇戶外平面停車場、觀光停車場、道之驛、景點自有停車場。
特別注意博多車站、運河城、百貨停車場先確認限高。
柳川可查市營本城町／筑紫町觀光停車場，或使用船公司指定停車場。
由布院金鱗湖周邊路窄，建議先找戶外停車場再步行。
