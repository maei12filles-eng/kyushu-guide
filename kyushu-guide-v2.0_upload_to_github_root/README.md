# 九州親子自駕旅行手冊 v2.0

這是 GitHub Pages 用的乾淨專案版。

## 檔案結構

- `index.html`：網頁內容與旅遊資料
- `css/style.css`：版面與視覺樣式
- `js/script.js`：選單、照片燈箱、行程渲染、Checklist 等互動功能
- `images/`：住宿照片

## 更新方式

1. 修改後上傳/覆蓋對應檔案。
2. GitHub Pages 會自動重新部署。
3. 網址維持不變：`https://maei12filles-eng.github.io/kyushu-guide/`

請保留首頁檔名為小寫 `index.html`。
